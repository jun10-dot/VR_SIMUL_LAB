# VR_SIMUL_LAB
VR 화학 실험실 안전 교육 시뮬레이션


## 📍 프로젝트 소개
가상현실 환경에서 화학 실험실의 안전 수칙을 체험하고 학습하는 교육용 시뮬레이션


## 🔎 프로젝트 정보
- 프로젝트 기간 : 2025년 11월 ~ 11월 (2주)
- 본인이 맡은 기간 : 3일
- 개발 인원 : 4명

## 담당 역할 (Technical Lead)
- 용액이 담긴 실린더를 비커에 혼합하는 과정
- 위험 용액을 혼합하면 가스,폭발 파티클 생성

## 담당 스크립트
- Beaker.cs
- BeakerDetector.cs
- Cylinder.cs

## 🦾 기술 스택
- 언어 : C#
- 엔진 : Unity Engine
- 플랫폼 : Meta Quest 기반 VR
- 시각 효과 : Unity Particle System

## 주요 기능
- VR 환경 내 실험 기구 조작
- 실험 결과 확인
- 안전 교육 안내 문구
